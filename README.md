# ThickV-Stain in PyTorch
A multi-scale virtual staining model for unprocessed thick tissues

## Prerequisites
- Linux or macOS
- Python 3
- CPU or NVIDIA GPU + CUDA CuDNN

## Getting Started
### Installation
- Clone this repo:
```bash
git clone https://github.com/TABLAB-HKUST/ThickV-Stain.git
cd ThickV-Stain
```
